# 不容乐观的世界

10月发生了很多事

中东打起来了 大人物去世了

秋季震撼没有如期而至 但是三千点准时的破了 世界大概是变得更坏了

如果对标大约一百年前的世界 那么现在应该是1932-1933年

毕竟外面这环境都这么糟糕了 希望明年能找个对象吧 希望......

# 没有人跟我很熟

明天的最低温度将达到1度

很难想象说仅仅一周以前的上周日 最高温度还是27度

夏天到冬天的速度快的有些离谱 但这是南京 这也正常


社恐倾向越来越明显

除了少数特定的有交集的人以外 对其他人没有任何想去接触的想法

不要觉得谁谁谁跟我很熟 现在是毕业后的第三年

除了少数个别以外 没有人跟我很熟


怀念曾经躺在宣城新港国际的大草坪上

晒着暖暖的太阳 小憩一会儿

那种舒适的 躺平的 非常自在的感觉了

可惜回不去了

# 无无又聊聊

最近的生活主打一个无聊又停滞，就像刚才打的那把王者，风暴龙王两边加起来打了三次，拉扯了半个小时，才算把对面干掉。又菜又混是这个样子，打农不行，打工也不行。打农在钻石慢慢混分，打工在14级慢慢混钱。

对v的兴趣渐渐没有之前那么大了，将近三十块粉丝牌已经灰了二十多块了。看v基本上也就当个背景版摆在那，然后干点什么别的事情。不发弹幕，不送礼物，纯纯黑听。前几年的群v混战也差不多告一段落了，现在剩下的v基本上半死不活的，也就变成了套皮游戏主播罢了。

看東雪莲一个切片，感觉很有道理。原话大概是：你都上一个星期上了五天班了，已经被生活拷打了，我还要在这里跟你说什么，调休啊，年终奖啊，生活的苦痛啊，赚多少钱，我玉玉了，叽里呱啦啊。然后你一听发现woc我们两个都是活在现实里的人，那谁救谁啊。那迪士尼公主是真的迪士尼公主吗？那玲娜贝儿里面的人是真的玲娜贝儿吗？就是世界上有很多好的谎言，虽然有很多不好的谎言不应该的，但大部分也有好的谎言。我们需要学会去接受，没必要个个都拆穿，搞得好像很不识趣一样。

生活大概就是这么回事，平平无奇，带点苦痛，带点玉玉。所以有的人总是想逃避现实，去提瓦特，去仙舟罗浮，或者什么罗德岛之类乱七八糟的地方。但随着赛博养胃的加剧，手机里装了一堆的游戏，也经常一个都不想玩。电脑新配了曲面大屏，但好久没开过机了。抽了芙芙练了几天练不动了，抽了藿藿就根本不想练了。

又一次感冒了，鼻塞流涕嗓子痛。今年好像感冒不知道多少回了，已经渐渐习惯了。但是习惯不等于接受，难受还是难受的。

那么，晚安。

# 跨江而行

夜幕下的江北大道高架路，如每一条城郊高架一般，车流滚滚，但杳无人迹。从浦口老城的浦东路拐入，三河桥、顶山、石佛寺、七里桥。道旁一座座高楼拔地而起，但更多的还是荒野和农田。这是所谓的江北核心区，但给我的感觉跟上海远郊的崧泽大道并没有多少不同。

在上海坐了无数次横跨黄浦江的轮渡，而南京的轮渡似乎也与此并无不同。花上两元，等待十到二十分钟不等，花十分钟躲避各种各样的船舶抵达彼岸。南京的宁浦线轮渡大概类似于上海的塘董线轮渡，一边是繁华城区的边缘(中山码头/董家渡)，另一边是郊区的破落老镇区(浦口/塘桥)。不远处的大桥作为江面上最早的桥梁(南京长江大桥/上海南浦大桥)抢走了大部分的客流，只剩下一些旅游客和骑行人作为渡口的常旅客。与之不同的是，南京长江大桥允许骑行，虽然说桥的高度骑行也够痛苦了。

小红书搜索枫叶，除去栖霞山之外的结果是一堆对莫愁湖的推荐。寻思也不是很远，就去转了一圈，就看那个院子里唯一的一棵枫树被一群人围着。前两天同事说现在小红书可以当百度用，不得不承认搜索结果的靠谱程度似乎也跟百度看齐了。

# 宁杭速通

按理来说，都到宣城了，多少应该回学校看一眼。但是学校里的人已经跟我没啥关系了，20那群的不是考研就是保研总归挺忙的，21往下的又没啥特别熟方便打扰的。再加上现在的门禁也已经不是我可以随便进去的了，所以回去属实没必要。

南京市中心到杭州市中心的距离是240km，但从南京最南的高淳宣狸路苏皖省界到杭州西北的临安千秋关皖浙省界，距离只有100km。发源于千秋关附近云梯乡的东津河一路顺流而下，经仙霞、中溪、梅林，在宁国城区汇入水阳江，再沿港口、水东、孙埠进入宣城市区，向北走五星、养贤、朱桥、水阳，经高淳的各条水系汇入南漪湖、固城湖、石臼湖，或转向芜湖流入长江。

溯溪而上，水与山的融合，构织出皖浙天路的绝美景色。云层渐低，如梯级般层层叠叠，是为云梯。过千秋关向临安，一路则是沟谷盘旋。我对于杭州最大的欣赏，便在于浙西的千山万水，与滚滚东流的钱塘江。但是，也仅止于此了。
