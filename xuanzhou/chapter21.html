# 溯世而来

# 明暗

阳光洒落 在大地上制造出一道长长的影子

你看着它 仿佛以为自己的腿真的有这么长

夕阳西下 逐渐黯淡的亮光将影子悄悄带走

你才发现 真正的自己是多么的渺小和卑微

# 虚拟生灵

我只是一只小嵩许 在宽阔的马路上瑟瑟发抖

为什么认识的人都那么强大

我只能隐藏在世界的边缘 远离人群

做一个只活在网络里的渣

学又学不下去 玩也玩不安心

可我也不是海王 我也不够热情

甚至也不想交多少朋友

看着代码的黑白和excel的鲜绿

所谓的小姐姐也不会去奔现

终究只是存在于电脑屏幕上的名字

或者硬盘里的一条毫无意义的数据

# 生而程序

生命的意义在二十岁上下最为怅惘

溯世之书 溯世而敛

叹过尽千帆后 凡生怎般

然千帆未历 又怎知那世间

生活本质是个程序 函数预留了一个对象的接口

但是每次传入的参数都没有

我也不清楚函数运行 到底需不需要这样一个对象

会不会让程序运行的更好

但获取对象的程序从未成功运行 中断或者异常

甚至需要回溯版本

既然如此 倒不如不执行

虚空广播 随缘抓取

为什么不自己创建？我没有类的声明

# 狱中的期待

终于确然的让大四没课了

我觉得应该也不会有什么后果 大不了再来一年

反正估计也要二战的

今年很多玄学的操作让我很迷

考试大纲莫名的变换让数学变成赌场

统考专业课不同不准调剂基本堵死了后路

拿到了大学生涯可能是最后的几张奖状

却已经失去了所有的意义和价值

校区一圈无形的铁窗配着有形的高墙

不远处的金陵一波未平一波又发

罢了 蹲在监狱里

期待大学的最后一个生日

虽然这期待看起来也不是那么必要

# 舍本逐末

被各路院校牛鬼蛇神的操作弄的心态有些不好

招十个人考试科目十三种

计算机专业必考信号系统

这就是21世纪20年代的考研吗

再结合前两天的大纲变动

也许这就是20岁的人需要适应的

来自社会的全方位无死角毒打吧


朴实无华而又枯燥的实验室社畜生活

似乎好像又迈进了新的一岁

看起来每天都在好好的复习 却不过大脑

就像洗衣机正常的工作了 但电机不转

# 二十岁

指针在秋的晚风里划过新的链表

迷失在无向图间的是节点的记号

一切的烦恼在死锁之中不肯谦让

感情的页表里也全是空白的字样

ALU里满满全是不明不白的念想

数据总线传递着都是心动的信号

在那路由器所能广播到的世界里

子网掩码掩盖的心终究无法送达

二十岁了 感谢今天陪伴我的所有人

第一次也是最后一次盛大的生日

是合工监里最终难舍的温柔

# 前浪·后浪

后浪在军训 后浪在转专业 后浪在。。。

前浪在考研 前浪在找工作 前浪在。。。

没有前浪也没有后浪 只有新鲜的韭菜和韭菜苗

不管是转专业成功的 还是苦涩的留在原专业的

都只不过走过了一个很小的路口

前方的荆棘依然很多 我只能祝你们幸福

# 夏天的风

夏天的风我永远记得/清清楚楚的说着爱我

我喜欢你暖暖的笑容/是得不到的温柔

独坐西山的日子一不小心就过去了20天

少了新安学堂无处不在的烦躁和压力

考研过程又回到了他按部就班的路上

成功不易 那便在最后的时间里 过的开心一点

又想起了遥远的三年多前 高三那段岁月

那时我也不曾多么的努力 也不会感动自己

除了刷题 生活更多的还是小说和游戏

或者在深夜的南京城遍历 晚风和夜的气息

渺茫的前途氤氲在教育部教学司函38号文里

信心也在无尽的等待里销声与匿迹

没有期待的话 那就去试图去逃避吧

卷是因为所有人都抱着更好的期待

我没有期待 是不是就可以不卷呢

按部就班的人失去了自我 成全了集体

独处一隅之人放弃了集体 也失去了自我

而真正自在的人 能够把集体融合为自我

很难说谁过的更成功 但我必定不够快乐

夏天的风曾暖暖吹过/穿过头发吹进耳朵

我拥有的夏天/只是风罢了

# 秋分

入秋后的第一杯奶茶 依然是来自合肥的问候

咸鱼般的“考研”人 不配拥有梦想和希望

大佬们的前途越高远 越让我看穿自己的弱小

几个月前的我曾经说过

如果坚持到报名都没有改变目标 那就已经成功了一半

起码有了想要成功的勇气

但是事实上看起来 这个成功的一半 有些难

但不管怎么说 总归还是有退路的

13号线能到知春路 也能到西二旗

可是最后最大概率的结果其实是 我哪也没去成
