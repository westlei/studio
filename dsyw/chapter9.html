# 向阳而生

# 青·嘉·苏

长三角的本质是江浙沪，而江浙沪的交点是青浦。江浙沪发达，但江浙沪的交点很贫穷。无论是青浦的金泽练塘，还是吴江的黎里盛泽，再或是嘉善的姚庄西塘，都是贫穷而只能发展旅游的乡村，甚至房价也远低于周边，上海金泽的房价甚至不如崇明，而其他几个地方则更加糟糕。当一年后的华为来到这里，又会对这里带来怎样的改变？大约，除了房价上涨，什么也不会有。

很难想象，在2023年，上海旁边的嘉善，公共交通依然是满地破中巴，而乡镇的面貌比起十年前似乎也没啥改观。嘉善县县城中心的街道也看不到什么人，河道边上依然是乡村自建房的感觉。主城道路到处都在修，遍地是坑坑洼洼的一车道临时便道。

嘉兴市区遍地的五芳斋，然而却是像永和豆浆一般的快餐店，价格高味道还不怎么样。市中心的月河古街、建国路一直到南湖大约是为数不多的门面，但是coco都已经活到了需要靠买一送一搞促销的地步。有轨电车速度属实不太行，行驶也不够平稳，好在客流似乎还可以。

当公交跨越省界进入吴江，终于看到了稍微现代化一些的城市风貌。吴江虽然经济条件欠佳，但是在三省交界的区域之中，大约是城市建设最好的一个。四号线察院场一小时直通，然后在奥林清华或吴江汽车站有专线一小时内通达所有乡镇。吴江主城西部的苏州湾区域引进了砂之船万象汇吾悦与爱琴海，仿佛深圳的欢乐港湾一样繁华。

# 嘉兴路口

从外滩向北两公里的虹口，有一个被称为上海音乐谷的3A级风景区。从4号线与10号线交汇的海伦路站出站，拐过两个弯，就能看到标着上海音乐谷的入口。说是音乐谷，但其实只不过是一个滨水风貌的石库门建筑群，有一些音乐创意园坐落其间。整个景区里最知名的景点大约是嘉兴路267号的星梦剧院，以及几个可以直接拍到东方明珠的网红打卡点。

星梦剧院的表面有些破败，与门口凌乱的电线和老式广告牌一起构成了一种八十年代小剧场的感觉。它的主人是旁边不远处创意园里一家叫丝芭文化传媒的小公司，也只服务于其旗下的知名女团SNH48。没有公演的日子，剧场附近十分寂静，只有少数几个前来打卡的游人。无数人的梦开始于这里，也结束于这里，亦或是结束于三公里外的虹口区法院。

河粉大约是个圈地自萌的团体，似乎极少在社交媒体上看见河粉的踪迹。但年年总选营收破亿的成绩，大约也说明了这个团体的庞大。十年的经营并没有为塞纳河带来多少的正面流量，屡屡破圈的反而是一些成员的负面新闻以及成员与公司对簿公堂的场面。说起塞纳河，一般路人印象最深的或许也只有鞠婧祎，以及九年前尴尬的四千年热搜。

作为一般意义上的普罗大众，我对SNH的印象同样仅限于此。如果还有点什么别的，那可能是高考后的时候有一档现在看来非常难评的综艺叫《我爱二次元》，从中认识了卡子哥(李艺彤)(ps:认识周深也是因为这档节目)。这大约也构成了我若干复杂成分之中与塞纳河有关的两个组成部分：蜜橘+卡推。当然还有些别的，那是后话。

作为爱好正常的标准直男，成为鞠骑首先自然是因为颜值，但是正经路转粉应该是从《叹云兮》开始的。作为声控高于颜控的歌粉，小鞠的音色实在是非常戳我。后来的《天长》《孤独与诗》和《冬日》，都是来来回回单曲循环N遍的神曲。至于卡子哥，更多的大约是比较有意思，毕竟正统偶像里走这个风格的不多。当然后来卡子哥走向元宇宙，那是另一个故事了。

关于塞纳河的下一个成分大约是脆鲨，大概是某段时间被贴吧铺天盖地的010刷屏洗脑，于是正儿八经开始关注七海了，抽象嘴臭观众对线什么的最好了。讲道理嘉然星瞳什么的实在看不来，幼教主播是真的无聊，看看東雪莲不好吗？生活越不如意，越喜欢看皮套；越看皮套，人就越抽象：人越抽象，生活就越不如意。但我都这样了，我不看皮套我看啥？

随着四禧丸子加入v圈江湖，塞纳河也彻底沦为了v圈培训基地。随着一个又一个塞纳河前偶像走上虚拟赛道，日v终究不得不开始退出江湖。猫雷毕业转生，再毕业再复活。lulu拉夫营业，然后入府开炒。不管大家怎么批判府狗，但是府系主播还是不错的，像萌鸽传媒的孙笑川258就很好。听说有本科同学马上都要结婚了，算算年纪二十四五岁也差不多了，而我对象呢？哦，管人痴哪里会有对象啊，你是女生你也不想你对象天天对着二次元小人发癫吧。

# 最不想成为的自己

这段时间确实有点破防了。

一方面是工作本身的压力。当非常坑逼的需求和要求极其严苛的SE结合在一起的时候，很难没有一种实在干不下去的感觉。五月下旬的被动切团队，让我不得不开始学一门几乎完全不会的语言(C++)。随即就是深陷于一个离谱的坑需求，让我产生了一种被团队边缘的感觉。新来的领导又跟我完全没有过交集，让这种焦虑变得更严重了。

另一方面是发展前景的灰暗。从五月开始的芯片大寒潮迅速刮过整个海思，附近的同事一个接一个离职或者被输出。另外顶层对部门的定位发生了很大的改变，整个部门的研发人员确定要被踢出到其他部门。考虑到当前这个团队几乎全是新员工，能力实在是不足，很难不怀疑当前团队会成为牺牲品。

一度觉得考研是一条出路，但是在跟很多人交流过以后发现这不是一个好的选择。先不提读研之后需要面对的导师压榨，单纯读研后进所这个想法本身就充满了问题。工作更辛苦但是钱更少，别人接受不了这种落差，我肯定更接受不了。所谓的稳定，在金钱的巨大落差面前，实在是不值一提。能接受收入降幅超过50%的人可能有，但绝对不多。

然后就是转岗的问题。眼下的选择包括但不限于无线、云PaaS、终端云安工(南京)以及滚回老东家IT西向(深圳)，涉及到城市的选择、职业的发展、未来的规划。方向上主要是三个：嵌入式软件、web开发和安全，城市也是三个：上海、深圳以及南京。一部分朋友的告诫是频繁更换工作方向对简历来说不好，最好是维持嵌入式方向或者回去搞安全方向，所以首先排除云。我爸妈对于城市选择的意见是建议要么回南京要么去总部(深圳)，留在上海总归不上不下发展尴尬，因此也排除无线。

回深圳的话面对的问题在于怕被清算。厨子用他自己的生命告诉了全世界左右横跳的后果，而我回去要面临的问题如出一辙。轻的话是被钉在耻辱柱上，严重的话是一套背锅走人丝滑小连招。回南京的话则意味着前十几年关于离开南京的努力基本化为泡影，后面的人生主线就会是催找对象——催婚——催生等一系列非常传统的操作(某种意义上也可以称为继承家业的基本条件)。不管怎么说在外地飘着，起码摆烂的话父母管不着。这么一想，去年来上海的这一步真的是属于脑子过热了。职业生涯到目前为止陷入了一种完完全全的尴尬处境，同事评价我脑子坏了，我觉得还真是。

一不小心翻到了匿问我答板块，翻到了之前写的若干回答。只能说，我应该是已经变成了我最不想变成的那种样子了。

# 23年的路口

人生终究是迈进了第23个年头。

从某种意义上来说，我宁可没有经历这一年。失败的转方向，失败的绩效和年终，前后经历四个领导，唯一主导的工作还变成了一口大锅，最后当时所追求的情感寄托也化为了云烟。疫情的猛烈收紧和突然放开给我的身体状况也带来了沉重的打击，心率再也没法像疫情前一般在一个正常的区间里运行。身体机能开始崩坏，腹泻、呕吐、抽筋，一切曾经习以为常的现状都不再正常。

当现实的生活陷入破败，精神生活也会一样的陷入空虚。皮套乘虚而入，我成了忠实的管人痴。每天晚上打开b站对着若干vtuber直播间，寻求心灵的慰藉。热水器擦边，小唐人乱舞，罕见破大防，标志回旋舞。尽管实在没什么营养，但是在直播间失去素质，消失理智，扔掉脑子，也是一种属于管人痴的快乐。

一款二字开放世界冒险游戏在另一款六字手游出现之后迅速的过气了，虽然大半是它自己的垃圾剧情和地图问题。六字手游短暂的火了些日子，但终究归于平平无奇。电子ED的时光里，穿行于江浙大地的山水之中，寻求心灵的安平。上海是个好地方，但是并不是很适合我。

那么，下一站又该去哪里呢？


上海是一个精英感很重的城市。

什么是精英感，就是一种自内而外的高人一等的感觉。工作要体面，生活要体面，社交要体面，做人要体面。吃要本帮私房、精致日料，喝要手冲咖啡、精酿啤酒，玩乐要野外camping、city walk，普通打个工也得讲究gap hour。

当然抛开这些表面的东西不谈，每个上海人都会有一种发自内心的高傲，面对更优秀的人会有一种要比他更强的感觉，于是卷的飞起。但是身在卷中的人不觉得自己卷，觉得都是理所应当的，我讨厌卷而不自知的人。

有的人会说跟优秀的人一起工作会让自己变得更优秀，并且喜欢这种感觉，但是不包括我。我总觉得他们在有意无意的pua我，甚至写一些诸如希望我继续释放潜能提升自我的评价，使得我无从反驳。这种生活让我很不舒适，以致于明明白白的逃避工作。没有人喜欢工作，尤其是这种压力非常大的工作。

深圳也是一个奋斗风格的城市，但是人家的奋斗是明明白白的奋斗，不会有一种你不成功是因为你自己不行的感觉。深圳缺乏所谓的文化底蕴，在另一种程度上反而是一种好事，哪怕是一切向钱看，也比所谓的精英文化要好的多。我不是精英，但是钱还是有一些的。我也不喜欢深圳，但是相比于上海，还是深圳更好一些。

在上海无聊的日子里，我会在黄浦江的两岸随便走走，或者去虹口看看老上海的建筑，看看提篮桥，看看嘉兴路的星梦剧院。但如果，不考虑所有工作或者生活或者一切问题，那么其实我很想待在苏州。喜欢这个城市给我的感觉，喜欢所有有意与无意的遇见。

总之，累了。三次元生活太难了，想转生二次元了。
